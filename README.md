# Keyboard Smasher

A lightweight Windows keyboard macro built with Python + Tkinter.

## Features
- Custom key sequences
- Adjustable delay
- Clean custom UI
- Always-on-top window
- No console window

## Installation (Windows)

### Option 1: Run from source
1. Install Python 3.10+
2. Clone the repo:
```bash
git clone https://github.com/YOUR_USERNAME/keyboard-smasher.git
cd keyboard-smasher
